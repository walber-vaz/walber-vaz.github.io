Projetos Práticos Desenvolvidos

Ao longo da minha jornada de aprendizado, desenvolvi diversos projetos práticos utilizando apenas HTML, CSS e JavaScript puro. Esses projetos incluem exercícios acadêmicos, páginas web responsivas, aplicações interativas e desafios propostos em disciplinas do curso. Todos os trabalhos foram realizados sem o uso de frameworks ou bibliotecas externas, focando no domínio das tecnologias fundamentais da web.

Se quiser conhecer mais sobre meus projetos, acesse meu GitHub: github.com/walber-vaz ou entre em contato pelo formulário disponível na página de contato.
